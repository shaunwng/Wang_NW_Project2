# Wang_NW_Project2
This repo is for 1056 Project 2 - Web Typography created by Naigong Wang.

This repo will demonstrate the ability to selecting appropriate fonts (using the techniques covered in
class i.e. google fonts or CSS3 @font-face) and CSS styles to enhance the copy. Line Height,
Leler Spacing, Font Weight, Colour and Type are the key components to this assignment.


## Prerequisites
All you need is use computer browser with an Internet connection.

## Usage
Check the main branch, clone the project, cd to your directory/ download and You're good to go!
